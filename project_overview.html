Project Overview

Topic

Kaleid AI’s project Reflections is a data visualisation application that displays journal entries as stars in the night-sky, with multiple journal entries linking together creating constellations. Reflections design concept is to visualise the user’s reflective thoughts by encouraging mental health, self-care and mindfulness. The team behind this vision has set out to reduce data intimidation.

Here are the Features:

  • Data Visualisation
  • Collation & Automatic Sharing
  • Dataset Search Engine
  • Dataset Analysis
  • Dataset Social Media
